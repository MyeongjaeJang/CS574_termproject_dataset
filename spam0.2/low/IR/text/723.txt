قابلیت های کلیدی نرم افزار Jitsi: - برقراری ارتباطات صوتی و تصویری امن- پاسخ دهی و فوروارد خودکار پیام ها- ضبط تماس- چند پروتکله بودن با پشتیبانی از gtalk و یاهو چت و ... - مالتی پلتفرم- رمزنگاری تماس با SRTP و ZRTP- برقرای تماس های ویدئوکنفرانس- برقراری ارتباطات چند رسانه ای مستقیم با استفاده از پروتکل ICE- رمزگذاری قدرتمند- انتقال فایل برای XMPP، AIM / ICQ، Windows Live Messenger، YIM- رمزگذاری پیام ها با OTR- پشتیبانی از IPv6 برای SIP و XMPP- تماس های صوتی و تصویری برای SIP و XMPP با استفاده از H.264 و H.263 یا VP8 برای رمزگذاری ویدئو- پشتیبانی از تماس های ویدئویی گروهی - رایگان، اوپن سورس و سازگار با WebRTC- و ... 
Jitsi (formerly called SIP Communicator) is a powerful, scalable, secure (using the SIP/RTP/SRTP/ZRTP encryption algorithms), high-quality and community-driven VoIP/video conferencing/instant messaging application from France, written in Java. It’s free, open-source, cross-platform (across Windows, Linux, Mac OS X desktop platfroms and Andrjoid, iOS mobile devices), multi-protocol and multi-user on use and allows to be made secondary development.As an audio/video chatting tool that works with all popular communication protocols (OTR, SIP, XMPP/Jabber, AIM/ICQ, IRC, Yahoo!, Windows Live/MSN, Facebook, GoogleTalk, etc.), Jitsi also supports online multiuser voice/video streaming and conferencing, and some other practical features such as chatting, desktop sharing, file transferring, and instant messaging. And, more importantly, it allows users to make this happen through many different protocols within the same software. Jitsi basically realized most functionalities of the converged communication, which is enough to meet the basic needs of small and medium-sized enterprise users. For example: the open-source Unified Communications PBX software Issabel has integrated Jitsi’s video meeting functionality.Undoubtedly, Jitsi is currently the most fully-functional and advanced audio/video communicating tool for Windows, Unix-like OS (Linux, Mac OS X, BSD, etc.) and mobile (Android, iOS). Whether it’s suitable for you or not, just give it a try. 
Here are some key features of "Jitsi": - 100% free, open source, and WebRTC compatible- Attended and blind call transfer- Auto away- Auto answer and Auto Forward- Call recording- Call encryption with SRTP and ZRTP- Conference calls- Direct media connection establishment with the ICE protocol- Desktop Streaming- Encrypted password storage using a master password- File transfer for XMPP, AIM/ICQ, Windows Live Messenger, YIM- Instant messaging encryption with OTR (end-to-end encrypted)- IPv6 support for SIP and XMPP- Media relaying with the TURN protocol- Message Waiting Indication (RFC 3842)- Voice and video calls for SIP and XMPP using H.264 and H.263 or VP8 for video encoding- Wideband audio with SILK, G.722, Speex and Opus- DTMF support with SIP INFO, RTP (RFC 2833/RFC 4733), In-band- Zeroconf via mDNS/DNS-SD (à la Apple‘s Bonjour)- DNSSEC- Group video support (Jitsi Videobridge)- Packet loss concealment with the SILK and Opus codecs More info (open/close)
